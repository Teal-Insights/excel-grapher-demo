# Repositories for the Nature Finance contract

**fastpyxl** - Fast, performant drop-in replacement for openpyxl (infrastructure) - https://github.com/Promptly-Technologies-LLC/fastpyxl
**excel-grapher** - Core Excel graph extraction and formula evaluation engine - https://github.com/Teal-Insights/excel-grapher
**lic-dsf-programmatic-extraction** - Pipeline that uses excel-grapher to turn the LIC DSF into a standalone library - https://github.com/Teal-Insights/lic-dsf-programmatic-extraction
**lic-dsf-2025-08-12** - Standalone Python library version of the LIC DSF 2025 template - https://github.com/Teal-Insights/lic-dsf-2025-08-12
